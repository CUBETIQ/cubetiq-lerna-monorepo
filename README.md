# CUBETIQ Lerna Monorepo

-   Base SDK
-   Basic SDK

### Usages

-   Bootstrap

```shell
pnpm bootstrap
```

-   Run Dev

```shell
pnpm dev
```

-   Run Build

```shell
pnpm build
```

- Clean Build

```shell
pnpm clean
```
